항공권 특가 알림 서비스

Pain Point :

  • 저렴한 항공편을 찾기 위해서 매일 웹사이트를 수동으로 확인
  • 많은 시간이 걸림
  • 가격 모니터링은 귀찮다.

사용자수 : 백만 명+

  • 월별 항공편 검색 건 : 450,000건+
